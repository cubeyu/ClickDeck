Privacy Policy for ClickDeck

Overview

ClickDeck is a browser extension that allows users to visually select, edit, and export HTML elements on any webpage. This privacy policy explains how the extension handles user data.

Data Collection

ClickDeck does not collect, store, or transmit any user data.

All operations — including element selection, style editing, text modification, and page export — are performed entirely within the user's browser. No data is sent to any external server.

Data Storage

The extension uses chrome.storage to save user preferences (such as language settings) locally in the browser. This data never leaves the user's device.

Third-Party Services

ClickDeck does not integrate with, transmit data to, or rely on any third-party services.

Permissions

The extension requests the following browser permissions:

  • activeTab — to access the current page when the user activates the extension
  • scripting — to inject the editor UI into the active page
  • storage — to save user preferences locally
  • downloads — to save edited pages as HTML or PDF files to the user's disk

These permissions are used solely for the extension's core functionality and are not used to collect or transmit any user data.

Changes

If this privacy policy is updated, the changes will be reflected on this page with an updated effective date.
