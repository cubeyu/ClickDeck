Privacy Policy for ClickDeck

Overview

ClickDeck is a browser extension that allows users to visually select, edit, refine, and export HTML elements on the current page. This privacy policy explains how the extension handles user data.

Data Collection

ClickDeck does not collect, store, or transmit personal page content to external servers.

All core operations — including element selection, style editing, text modification, visual edit suggestions, and page export — are performed locally in the user's browser.

Data Storage

The extension uses chrome.storage to save local preferences such as language or interface state. This data stays on the user's device.

Third-Party Services

ClickDeck does not integrate with, transmit data to, or rely on any third-party services.

Permissions

The extension currently requests the following browser permissions:

  • activeTab — to access the current page when the user activates the extension
  • scripting — to inject the editor UI into the active page
  • storage — to save user preferences locally

ClickDeck also declares host access so it can run on the page the user chooses to edit. These permissions are used only for the extension's core functionality and not for user-data collection or transmission.

Changes

If this privacy policy is updated, the changes will be reflected on this page with an updated effective date.
