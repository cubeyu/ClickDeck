# ClickDeck v1.2.1 Release Notes

ClickDeck v1.2.1 is a stability and compliance release focused on making the Chrome Web Store upload safe again while keeping the core editing experience intact.

## Highlights

- Removed the legacy browser-print PDF/debug path from the production package.
- Kept image-based PDF export for long pages, A4, and 16:9 layouts.
- Added a broader presentation compatibility layer for common HTML presentation hosts.
- Added diagnostics capture for complex HTML presentation pages.
- Updated the README download link to point to the `v1.2.1` release ZIP.

## What this release is for

- A safer Chrome Web Store resubmission.
- A cleaner release package for testers.
- A stable baseline before continuing deeper HTML presentation compatibility work.

## Installation

1. Download `ClickDeck-v1.2.1.zip` from the GitHub Releases page.
2. Extract the ZIP file.
3. Load the unpacked extension from Chrome or Edge.

## Notes

- This release keeps ClickDeck in MVP scope.
- Complex HTML presentation pages may still need future compatibility work.
- The release ZIP should contain only the built extension files.

## 中文说明

ClickDeck v1.2.1 是一次以“稳定性 + 商店合规”为主的发布版本，目标是恢复 Chrome Web Store 上架流程，同时保留当前核心编辑体验。

### 主要变化

- 移除了生产包中的旧版浏览器打印 PDF / debug 链路。
- 保留并继续使用图片型 PDF 导出，支持长页、A4 和 16:9。
- 增强了常见 HTML 演示宿主的兼容层。
- 新增复杂 HTML 演示页面的诊断采集能力。
- README 的下载链接已更新为 `v1.2.1`。

### 发布目的

- 重新提交 Chrome Web Store。
- 给试用者提供更干净的 release ZIP。
- 为后续更深的 HTML 演示兼容工作打基础。

### 说明

- 本版本仍然保持 ClickDeck 的 MVP 边界。
- 复杂 HTML 演示页面后续仍可能需要继续兼容。
- 发布 ZIP 应只包含构建后的扩展文件。
