ClickDeck

ClickDeck is a Chrome/Edge extension for visually fine-tuning the current HTML page in the browser. This page exists only to provide a stable public entry point and privacy policy URL for distribution and review.

What this page is for

It is not a product landing page. It is a minimal public page used to host the ClickDeck privacy policy and provide a stable reference URL for Chrome Web Store review.

Public product files, release packages, and development materials continue to live in the GitHub repository and release assets. This page intentionally stays minimal.
